## Variables

### Déclaration:

```c
int nb; //déclaration d'un entier
float nb; //déclaration d'un floattant
char nb; //déclaration d'un caractère
```

### Initialisation:

```c
int nb=0; //exemple d'initialisation d'une variable
```
